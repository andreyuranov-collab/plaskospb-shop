# PLASKO SPB shop v00.12 test 260709

Версия с корректными карточками **PLASKO PLA Standard 1.75 мм**.

Источник данных: сохранённая страница оригинального сайта PLASKO из архива `260709.zip`.

Перенесено 1:1: наименование товара, цвет, описание, диаметр, вес, точность, розничная цена, оптовая цена.

Изображения сопоставлены с файлами `PLA_ST_*.jpg`. Для позиции «Золотистый» в сохранённой странице оригинального сайта нет отдельного `<img>`-файла — оставлена карточка с цветовой визуализацией, как в исходной странице.

## Загрузка в GitHub
1. Распаковать архив.
2. Зайти в папку `plaskospb_shop_v00_12_test_260709`.
3. Загрузить **содержимое папки** в корень GitHub-репозитория.
