# PLASKO SPB shop v00.21 test 260714

Версия собрана на базе v00.20 без изменения исходных изображений и структуры фотоблока PLASKO.

## Изменение

- ширина сетки каталога приведена к масштабу оригинального сайта PLASKO;
- сетка использует `width: min(1200px, 88vw)` и центрируется;
- боковые поля каталога соответствуют примерно 6% ширины окна с каждой стороны;
- карточки PLA и ABS стали уже, поэтому исходные фотографии должны отображаться в том же масштабе, что на оригинальном сайте;
- PETG, изображения, товарные данные, фильтры и корзина не изменялись;
- временная служебная строка версии в футере сохранена.

## Загрузка в GitHub

2. Открыть папку `plaskospb_shop_v00_21_test_260714`.
3. Загрузить **содержимое папки** в корень репозитория.
