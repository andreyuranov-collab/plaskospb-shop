# PLASKO SPB shop v00.20 test 260714

В этой версии структура размещения изображений повторена по сохранённой странице оригинального сайта PLASKO без собственной интерпретации.

Перенесено один в один:
- `catalog-grid`;
- `product-card product-card--clickable`;
- `product-img`;
- два вложенных `div` внутри фотоблока;
- исходный inline-style изображения;
- `aspect-ratio: 3/4`;
- `max-height: 350px`;
- `object-fit: cover`;
- исходные необработанные изображения поставщика.

Не используются физическая обрезка, производные изображения, `object-position`, фиксированная высота 420 px или собственный блок `art`.

Временная строка версии в футере сохранена. Перед релизом её нужно удалить.
