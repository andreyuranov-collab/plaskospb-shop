# PLASKO SPB shop v00.10 test 260707

Тестовая частичная версия сайта торгового представительства.

Что включено в этой версии:
- исходный логотип PLASKO из архива пользователя;
- добавление приставки `SPB` рядом с оригинальным логотипом без перерисовки самого логотипа;
- фотографии PLA Standart из пользовательского архива;
- частичное оформление карточек PLA Standart на главной;
- главная остаётся минималистичной и начинается сразу с каталога.

Что **не** включено в этой версии:
- полный ассортимент PLASKO 1:1;
- точные цены 1:1;
- окончательная товарная матрица.

Важно:
- каталог в этой версии является частичным предпросмотром по подтверждённым исходным материалам;
- новые товарные позиции, цены и названия не выдумываются;
- следующий этап — перенос полной матрицы 1:1 с оригинального сайта по мере получения исходных данных.

## Как загружать в GitHub
1. Распаковать архив.
2. Зайти в папку `plaskospb_shop_v00_10_test_260707`.
3. Загрузить **содержимое папки** в корень GitHub-репозитория.
4. Не загружать саму верхнюю папку в репозиторий.
