# PLASKO SPB Shop v00.01 test 260707

Статическая тестовая версия интернет-магазина для домена `plaskospb.ru`, подготовленная под схему GitHub → Cloudflare Pages по аналогии с сайтом ЛП.

## Что внутри

- `index.html` — одностраничный магазин: hero, каталог, фильтры, корзина-заявка, оптовая форма, доставка, контакты.
- `styles.css` — весь дизайн и адаптив.
- `script.js` — каталог товаров, фильтры, корзина, localStorage, формирование заказа через `mailto:`.
- `assets/` — SVG-логотип, favicon и hero-иллюстрация.
- `data/products.json` — справочная копия структуры каталога.
- `docs/DEPLOY.md` — инструкция по загрузке через веб-интерфейс GitHub и подключению Cloudflare Pages без командной строки.
- `_headers`, `robots.txt`, `sitemap.xml` — базовые файлы для хостинга и индексации.

## Важные пометки перед публичным запуском

1. В `script.js` заменить контакты в `siteConfig`:
   - `phoneDisplay`
   - `phoneHref`
   - `email`
   - `city`, если нужно.
2. В массиве `products` заменить тестовые цены, остатки, названия цветов и товарные описания на фактические.
3. Проверить юридическую информацию: продавец, ИНН/ОГРН/адрес, политика конфиденциальности, оферта, порядок оплаты и возврата.
4. Текущий механизм заказа — корзина-заявка через почтовый клиент. Для полноценной оплаты нужно добавить платёжный шлюз или CMS/CRM-интеграцию.

## Локальный просмотр

Для простой проверки можно открыть `index.html` двойным кликом.

Если браузер заблокирует чтение локальных данных, удобнее проверить сайт уже после загрузки на GitHub и первого деплоя Cloudflare Pages. Командная строка для загрузки на GitHub не нужна.
