# PLASKO SPB shop v00.16 test 260714

Изменения версии:
- увеличена высота фотоблока карточки до 420 px;
- фотографии центрированы так, чтобы катушка была видна полностью: без пустого поля сверху и без обрезки снизу;
- при переключении плашек материалов автоматическая прокрутка отключена;
- кнопка «В корзину» больше не открывает правую панель;
- после добавления товара на карточке появляются кнопки «−» и «+» с количеством между ними;
- правая панель корзины открывается только по кнопке «Корзина» в хедере;
- каталог, цены, фотографии и остальные данные сохранены без изменений.
